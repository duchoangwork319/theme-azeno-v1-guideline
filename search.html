Exclude products from Search

  1. Add tag to the product(s).

    You can update it directly in Product, or via export/import

  2. Update tag to exclude search setting.

    Go to Editor > Theme settings > Search Config. Add the same tag from product to the text box.

    NOTE: This will affect on: Related + Recently Viewed products on PDP, Search bar, Search page, Google search
