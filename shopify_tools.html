STEP 1: Prepare Product CSV

  1. Go to Admin page and export products that you want to sync.

STEP 2: Access Tool

  1. Access this link: Shopify Tools

  2. Import the Product CSV to the app.

    After importing, the app will show the CSV preview table.

STEP 3: Features

  1. Let's walkthrough the "Fetch" features

    • Publish products: This option will update Published="TRUE" and Status="active" after fetching data.

    • Inventory Policy Continue: This option will update Variant Inventory Policy="continue" to enable Continue selling when out of stock feature, which allows customers to purchase the Sales products even when they are out of stock

    • Handle Suffix: This option will automatically add a suffix value to product Handle after fetching data if you want to duplicate products for special occasion sales.

    • Append Tags: This option will automatically add tags to product Tags after fetching data if you want products to be added to smart collection(s).

STEP 4: Download

  1. After fetching successfully, the Download button will be enabled for you to download the new CSV file.

STEP 5: Rollback and Detach

After fetching successfully, You have 2 ways to revert table to your inital data.

  1. Click on Rollback button.

    It will rollback you table to the inital stage and remove data for download feature.

  2. Click on Detach button.

    It will remove the file from the app and reset all data. You can use this to import another CSV file.

NOTE

The app will not sync these columns below to avoid importing incorrect data from another source.

  1. Variant Inventory Qty
  2. Variant Price
  3. Size Chart (product.metafields.bwp_fields.size_chart)
